# Chrome Web Store 提出情報 — X抽出 (xt)

## 1. ストアリスティング情報

### 基本情報

| 項目 | 値 |
|------|-----|
| **名前** | X抽出 (xt) |
| **バージョン** | 1.0 |
| **カテゴリ** | Productivity |
| **言語** | Japanese |

### 短い説明 (132文字以内)

```
X上のツイートをタップ選択してURLを一括コピー。ブックマーク・タイムライン・検索結果どこでも使える。
```

(48文字 — 制限内)

### 詳細な説明 (ストアページ用)

```
X (Twitter) 上のツイートURLを素早く収集するための拡張機能です。

■ 使い方
1. X (x.com / twitter.com) を開くと、画面下部にフローティングバーが表示されます
2. ツイートをクリックして選択（青くハイライトされます）
3. 「コピー」ボタンで選択したツイートのURLを一括コピー

■ 便利な機能
・「表示中を選択」— 現在画面に見えているツイートを一括選択
・「直近50件」— 読み込み済みの直近50件を一括選択
・「全解除」— 選択をすべてクリア
・✕ボタンでバーを非表示にして通常ブラウジングに戻れます

■ どこでも使える
・タイムライン
・ブックマーク
・検索結果
・ユーザープロフィールのツイート一覧
・リスト
X上のarticle要素がある全てのページで動作します。

■ プライバシー
・個人データの収集・送信は一切ありません
・全ての処理はブラウザ内でローカルに完結します
・外部サーバーとの通信はありません
```

## 2. 必要なアセット

### アイコン (必須)

| ファイル | サイズ | 状態 |
|----------|--------|------|
| `icon16.png` | 16x16 px | **要作成** |
| `icon48.png` | 48x48 px | **要作成** |
| `icon128.png` | 128x128 px | **要作成** |

デザイン案: 青い円 (#1d9bf0) の上に白文字で「xt」。シンプルで視認性重視。

作成方法の例:
```bash
# ImageMagickがインストール済みの場合:
convert -size 128x128 xc:none \
  -fill '#1d9bf0' -draw 'circle 64,64 64,2' \
  -font Helvetica-Bold -pointsize 56 -fill white \
  -gravity center -draw 'text 0,0 "xt"' \
  icon128.png

convert icon128.png -resize 48x48 icon48.png
convert icon128.png -resize 16x16 icon16.png
```

または Figma / Canva 等で作成。

### スクリーンショット (最低1枚、推奨3〜5枚)

サイズ: 1280x800 または 640x400

推奨スクリーンショット構成:
1. **タイムラインでツイートを複数選択している画面** — フローティングバーと青いハイライトが見える状態
2. **「コピー済み」表示** — コピー完了後のフィードバック表示
3. **ブックマークページでの使用** — どのページでも使えることを示す
4. **検索結果での使用** — 汎用性のデモ

## 3. プライバシー

### 単一目的の説明

```
X (Twitter)上のツイートURLをタップ選択でコピーする
```

### 権限の正当性

| 権限 | 理由 |
|------|------|
| `clipboardWrite` | 選択したツイートURLをクリップボードにコピーするため |

### ホスト権限

`https://x.com/*` と `https://twitter.com/*` — ツイートのarticle要素を検出し、選択UIを表示するため。

### データ使用の開示

- 個人データの収集: **なし**
- 外部への送信: **なし**
- 全ての処理はローカルで完結
- Chrome Web Store の Data use disclosure で全項目「No」を選択

### プライバシーポリシー

不要（個人データの収集を行わないため）。
ただし公開する場合は簡易的なものをGitHub Pages等に設置可能。

## 4. manifest.json の更新

`icons` フィールドを追加済み:

```json
{
  "manifest_version": 3,
  "name": "X抽出",
  "version": "1.0",
  "description": "X上のツイートをタップ選択してURLをコピー",
  "icons": {
    "16": "icon16.png",
    "48": "icon48.png",
    "128": "icon128.png"
  },
  "content_scripts": [
    {
      "matches": ["https://x.com/*", "https://twitter.com/*"],
      "js": ["content.js"],
      "css": ["style.css"]
    }
  ],
  "permissions": ["clipboardWrite"]
}
```
